Projects

A selection of projects showcasing my skills in data analysis, visualization, and behavioral insight generation.

Predicting Air Turbulence with Machine Learning

Due to climate change, incidents of air turbulence have been on the rise. This project uses Python to identify the key predictors of air turbulence and shows how air turbulence can be predicted using machine learning models.

Predicting Singapore HDB Resale Prices

This project uses Python to identify the key predictors of Singapore HDB resale prices and shows how HDB resale prices can be predicted using machine learning models.

Understanding U.S. Presidential Elections 2000-2024

This project uses Tableau to analyse electorate trends, voting trends, and party loyalty across states and counties for the U.S. presidential elections between 2000-2024.

U.K. Gender Pay Gap

This project uses SQL to analyse key trends in the U.K. gender pay gap report of 2021.

The Auditory English Lexicon Project (AELP)

The AELP is a multi-talker, multi-region psycholinguistic database of 10,170 spoken words and an identical number of nonwords. Analyses were performed in R.

LexiCAL

LexiCAL is a fast, simple, and intuitive calculator for lexical variables. It is a standalone executable developed using Python.
